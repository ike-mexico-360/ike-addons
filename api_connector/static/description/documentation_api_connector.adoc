== icon:info-circle[] Vue d'ensemble

Le module **API Connector** est un outil Postman-like intégré à Odoo qui permet de configurer et exécuter des appels API sortants de manière graphique et automatisée.

=== Cas d'usage principaux

[TIP]
====
* **Synchronisation données** : Envoyer des commandes, factures, contacts vers des systèmes externes
* **Intégrations e-commerce** : Synchroniser stocks, prix, commandes avec des plateformes
* **Notifications automatiques** : Alerter des systèmes tiers lors d'événements Odoo
* **Upload de fichiers** : Envoyer des documents (factures PDF, bons de livraison) vers des APIs
* **Webhooks sortants** : Déclencher des actions sur des services externes
====

== icon:sitemap[] Architecture du Module

=== Modèles principaux

[cols="1,2,3"]
|===
|Modèle |Nom technique |Description

|**Collections**
|`api.collection`
|Regroupement d'endpoints avec configuration partagée (authentification, URL de base, variables)

|**Connecteurs**
|`api.connector` 
|Configuration individuelle d'un endpoint API (URL, méthode, corps de requête, authentification)

|**Logs d'appels**
|`api.call.log`
|Historique détaillé de tous les appels API avec temps de réponse, statut, erreurs
|===

=== Hiérarchie et héritage

[source,text]
----
Collection (api.collection)
├── Configuration partagée
│   ├── Base URL
│   ├── Authentification
│   ├── Headers par défaut
│   └── Variables de collection
└── Connecteurs (api.connector)
    ├── Hérite de la collection (optionnel)
    ├── Configuration spécifique
    └── Variables d'endpoint
----

== icon:cog[] Configuration des Collections

=== Champs principaux

[cols="1,1,3"]
|===
|Champ |Type |Description

|`name`
|Char
|Nom de la collection (ex: "API Shopify", "Webhooks Stripe")

|`base_url`
|Char
|URL de base partagée (ex: "https://api.shopify.com/v1")

|`auth_type`
|Selection
|Type d'authentification : None, Basic, Bearer, API Key, OAuth 2.0

|`collection_variables`
|Text (JSON)
|Variables partagées utilisables dans tous les endpoints de la collection

|`default_headers`
|Text (JSON)
|Headers HTTP par défaut appliqués à tous les endpoints
|===

=== Variables de collection

Les variables de collection permettent de centraliser des valeurs réutilisables :

[source,json]

[NOTE]

[source,text]
----

== icon:magic[] Intégrations No-Code Serveur

=== Concept révolutionnaire

Le module API Connector permet de créer des **intégrations complètes sans écrire de fichiers Python**. Toute la configuration se fait via l'interface Odoo standard en combinant :

* **API Connectors** : Configuration graphique des appels API
* **Crons Odoo** : Planification via l'interface standard
* **Actions automatisées** : Traitement des réponses via l'interface

=== Workflow no-code serveur

[source,text]
.Processus en 3 étapes
----
1. 📡 API Connector → Appel API externe programmé
2. 🔄 Cron Odoo → Exécution périodique automatique  
3. ⚡ Action automatisée → Traitement de la réponse
----

==== Étape 1 : Configuration API (100% graphique)

[TIP]
====
* **URL et authentification** : Interface API Connector
* **Variables dynamiques** : Substitution automatique
* **Corps de requête** : JSON avec variables `{variable_name}`
* **Monitoring** : Logs automatiques intégrés
====

==== Étape 2 : Planification (Interface Odoo)

[source,python]
.Code minimal dans le cron (via interface)
----
# Configuration dans Paramètres > Technique > Automatisation > Actions planifiées
connector = env['api.connector'].browse(123)  # ID depuis l'interface
result = connector.execute_call({
    'date': datetime.date.today().strftime('%Y-%m-%d'),
    'company_id': env.company.id

----

==== Étape 3 : Traitement automatique (Interface Odoo)

[source,python]
.Action automatisée sur api.connector (via interface)
----
# Configuration dans Paramètres > Technique > Automatisation > Actions automatisées
# Déclencheur : "Lors de la mise à jour d'un enregistrement"
# Condition : last_response_success == True

import json

if record.last_response_data:
    response = json.loads(record.last_response_data)
    
    # Traiter la réponse selon le type d'intégration
    if record.name == 'Sync Products':
        for product_data in response.get('products', []):
            env['product.product'].create({
                'name': product_data['name'],
                'default_code': product_data['sku'],
                'list_price': product_data['price']
            })
    
    elif record.name == 'Import Orders':
        for order_data in response.get('orders', []):
            partner = env['res.partner'].search([('email', '=', order_data['customer_email'])], limit=1)
            if partner:
                env['sale.order'].create({
                    'partner_id': partner.id,
                    'amount_total': order_data['total']
                })
----

=== Templates d'actions automatisées

==== Template : Création d'enregistrements

[source,python]
.Action automatisée générique
----
# Condition : last_response_success == True and 'create_' in record.name
import json

if record.last_response_data:
    response = json.loads(record.last_response_data)
    
    # Mapping générique depuis endpoint_variables
    mapping_config = json.loads(record.endpoint_variables or '{}')
    target_model = mapping_config.get('target_model', 'res.partner')
    field_mapping = mapping_config.get('field_mapping', {})
    
    for item in response.get('data', []):
        values = {}
        for odoo_field, api_field in field_mapping.items():
            if api_field in item:
                values[odoo_field] = item[api_field]
        
        if values:
            env[target_model].create(values)
            _logger.info(f"Created {target_model} from API response")
----

==== Template : Mise à jour d'enregistrements

[source,python]
.Action automatisée de mise à jour
----
# Condition : last_response_success == True and 'update_' in record.name
import json

if record.last_response_data:
    response = json.loads(record.last_response_data)
    mapping_config = json.loads(record.endpoint_variables or '{}')
    
    target_model = mapping_config.get('target_model')
    search_field = mapping_config.get('search_field', 'id')
    
    for item in response.get('data', []):
        search_value = item.get(search_field)
        if search_value:
            existing_record = env[target_model].search([(search_field, '=', search_value)], limit=1)
            if existing_record:
                update_values = {}
                for odoo_field, api_field in mapping_config.get('field_mapping', {}).items():
                    if api_field in item:
                        update_values[odoo_field] = item[api_field]
                
                if update_values:
                    existing_record.write(update_values)
----

[source,python]
.Exécution en cron
----
# Cron quotidien de synchronisation
for order in env['sale.order'].search([('state', '=', 'sale'), ('synced', '=', False)]):
    connector = env['api.connector'].browse(123)  # ID du connecteur
    
    result = connector.execute_call({
        'product_variant_id': order.order_line[0].product_id.shopify_variant_id,
        'quantity': order.order_line[0].product_uom_qty,
        'unit_price': order.order_line[0].price_unit,
        'customer_email': order.partner_id.email,
        'customer_firstname': order.partner_id.firstname,
        'customer_lastname': order.partner_id.lastname,
        'creation_date': order.date_order.strftime('%Y-%m-%d')
    })
    
    if result.get('success'):
        order.write({'synced': True, 'shopify_order_id': result.get('data', {}).get('id')})
        _logger.info("Order %s synchronized to Shopify", order.name)
    else:
        _logger.error("Failed to sync order %s: %s", order.name, result.get('error'))
