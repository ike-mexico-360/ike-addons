== icon:info-circle[] Overview

The **API Connector** module is a Postman-like tool integrated into Odoo that allows you to configure and execute outbound API calls graphically and automatically.

=== Main Use Cases

[TIP]
====
* **Data Synchronization**: Send orders, invoices, contacts to external systems
* **E-commerce Integrations**: Sync inventory, prices, orders with platforms
* **Automatic Notifications**: Alert third-party systems on Odoo events
* **File Upload**: Send documents (PDF invoices, delivery notes) to APIs
* **Outbound Webhooks**: Trigger actions on external services
====

== icon:sitemap[] Module Architecture

=== Main Models

[cols="1,2,3"]
|===
|Model |Technical Name |Description

|**Collections**
|`api.collection`
|Group of endpoints with shared configuration (authentication, base URL, variables)

|**Connectors**
|`api.connector` 
|Individual API endpoint configuration (URL, method, request body, authentication)

|**Call Logs**
|`api.call.log`
|Detailed history of all API calls with response time, status, errors
|===

=== Hierarchy and Inheritance

[source,text]
----
Collection (api.collection)
├── Shared Configuration
│   ├── Base URL
│   ├── Authentication
│   ├── Default Headers
│   └── Collection Variables
└── Connectors (api.connector)
    ├── Inherits from collection (optional)
    ├── Specific configuration
    └── Endpoint variables
----

== icon:magic[] No-Code Server Integrations

=== Revolutionary Concept

The API Connector module enables creating **complete integrations without writing Python files**. All configuration is done via the standard Odoo interface by combining:

* **API Connectors**: Graphical API call configuration
* **Odoo Crons**: Scheduling via standard interface
* **Automated Actions**: Response processing via interface

=== No-Code Server Workflow

[source,text]
.3-step process
----
1. 📡 API Connector → Programmed external API call
2. 🔄 Odoo Cron → Automatic periodic execution  
3. ⚡ Automated Action → Response processing
----

==== Step 1: API Configuration (100% graphical)

[TIP]
====
* **URL and authentication**: API Connector interface
* **Dynamic variables**: Automatic substitution
* **Request body**: JSON with `{variable_name}` variables
* **Monitoring**: Integrated automatic logs
====

==== Step 2: Scheduling (Odoo Interface)

[source,python]
.Minimal code in cron (via interface)
----
# Configuration in Settings > Technical > Automation > Scheduled Actions
connector = env['api.connector'].browse(123)  # ID from interface
result = connector.execute_call({
    'date': datetime.date.today().strftime('%Y-%m-%d'),
    'company_id': env.company.id
})
----

==== Step 3: Automatic Processing (Odoo Interface)

[source,python]
.Automated action on api.connector (via interface)
----
# Configuration in Settings > Technical > Automation > Automated Actions
# Trigger: "On Update of a Record"
# Condition: last_response_success == True

import json

if record.last_response_data:
    response = json.loads(record.last_response_data)
    
    # Process response according to integration type
    if record.name == 'Sync Products':
        for product_data in response.get('products', []):
            env['product.product'].create({
                'name': product_data['name'],
                'default_code': product_data['sku'],
                'list_price': product_data['price']
            })
    
    elif record.name == 'Import Orders':
        for order_data in response.get('orders', []):
            partner = env['res.partner'].search([('email', '=', order_data['customer_email'])], limit=1)
            if partner:
                env['sale.order'].create({
                    'partner_id': partner.id,
                    'amount_total': order_data['total']
                })
----

=== Automated Action Templates

==== Template: Record Creation

[source,python]
.Generic automated action
----
# Condition: last_response_success == True and 'create_' in record.name
import json

if record.last_response_data:
    response = json.loads(record.last_response_data)
    
    # Generic mapping from endpoint_variables
    mapping_config = json.loads(record.endpoint_variables or '{}')
    target_model = mapping_config.get('target_model', 'res.partner')
    field_mapping = mapping_config.get('field_mapping', {})
    
    for item in response.get('data', []):
        values = {}
        for odoo_field, api_field in field_mapping.items():
            if api_field in item:
                values[odoo_field] = item[api_field]
        
        if values:
            env[target_model].create(values)
            _logger.info(f"Created {target_model} from API response")
----

==== Template: Record Update

[source,python]
.Update automated action
----
# Condition: last_response_success == True and 'update_' in record.name
import json

if record.last_response_data:
    response = json.loads(record.last_response_data)
    mapping_config = json.loads(record.endpoint_variables or '{}')
    
    target_model = mapping_config.get('target_model')
    search_field = mapping_config.get('search_field', 'id')
    
    for item in response.get('data', []):
        search_value = item.get(search_field)
        if search_value:
            existing_record = env[target_model].search([(search_field, '=', search_value)], limit=1)
            if existing_record:
                update_values = {}
                for odoo_field, api_field in mapping_config.get('field_mapping', {}).items():
                    if api_field in item:
                        update_values[odoo_field] = item[api_field]
                
                if update_values:
                    existing_record.write(update_values)
----

[source,json]
.JSON request body
----
{
    "order": {
        "line_items": [
            {
                "variant_id": "{product_variant_id}",
                "quantity": "{quantity}",
                "price": "{unit_price}"
            }
        ],
        "customer": {
            "email": "{customer_email}",
            "first_name": "{customer_firstname}",

        },
        "source_name": "{source}",
        "note": "Order created from Odoo on {creation_date}"
    }

[source,python]
.Cron execution
----
# Daily synchronization cron
for order in env['sale.order'].search([('state', '=', 'sale'), ('synced', '=', False)]):
    connector = env['api.connector'].browse(123)  # Connector ID
    
    result = connector.execute_call({
        'product_variant_id': order.order_line[0].product_id.shopify_variant_id,
        'quantity': order.order_line[0].product_uom_qty,
        'unit_price': order.order_line[0].price_unit,
        'customer_email': order.partner_id.email,
        'customer_firstname': order.partner_id.firstname,
        'customer_lastname': order.partner_id.lastname,
        'creation_date': order.date_order.strftime('%Y-%m-%d')
    })
    
    if result.get('success'):
        order.write({'synced': True, 'shopify_order_id': result.get('data', {}).get('id')})
        _logger.info("Order %s synchronized to Shopify", order.name)
    else:
        _logger.error("Failed to sync order %s: %s", order.name, result.get('error'))
----

# Modify an existing route
